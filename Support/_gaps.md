# Cevaplanamayan Sorular Logu (_gaps.md)

Bu dosya, Chat Agent'ın bilgi bankasında cevap bulamadığı soruları otomatik olarak kaydeder.
Doc Agent bu dosyayı okuyarak hangi kılavuzların oluşturulması gerektiğini belirler.

**Format:**
- `[AÇIK]` → Doc Agent henüz ele almadı
- `[KAPALI]` → Kılavuz oluşturuldu, soru çözüldü
