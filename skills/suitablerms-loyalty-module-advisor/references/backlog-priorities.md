# SuitableRMS Loyalty Backlog Priorities

Use this as the default product prioritization frame unless the user explicitly overrides it.

## Hemen Backlog'a Girsin

- Hazir segmentler
  `yeni musteri`, `aktif`, `uzun suredir gelmeyen`, `yuksek harcayan`, `sadakat uyesi`, `VIP`
- Lifecycle kampanyalari
  `hos geldin`, `2. ziyaret`, `3. ziyaret`, `dogum gunu`, `win-back`, `tier yukselince odul`
- Odul modelinin toparlanmasi
  puan, kupon, ucretsiz urun, ozel fiyat, indirim, damga mantigi
- Tier sisteminin guclenmesi
  seviye atlama, seviye avantajlari, gorunurluk, gerekirse yillik koruma/reset
- Omnichannel temel tutarlilik
  POS, masa/garson, kiosk, online, mobil
- Temel suiistimal kontrolleri
  tekil kupon kullanimi, manuel odul loglama, kritik islemlerde onay
- Hazir kampanya sablonlari
  pazarlama ve operasyon ekipleri icin

## Bekleyebilir

- Abonelik modeli
- Gelismis gamification
  streak, gorev, rozet, ilerleme, instant win
- Deneyimsel oduller
  VIP etkinlik, tadim, ozel erisim
- Daha zengin musteri profili
- Kampanya performans ekranlari
- A/B test
- Fis/QR ile offline-dijital kopruler

## Simdilik Dokunmayalim

- Probabilistic identity resolution
- Card-linking
- Receipt OCR
- AI text-to-rule
- ML churn prediction
- Next best action engine
- Hava durumu veya envanter bazli dinamik tetikleyiciler
- Ileri fraud analitigi
- 50ms altina inmek icin asiri erken teknik optimizasyon
